# Deep Whistle help images

Drop the screenshots referenced by `../deep_whistle_help.md` into this folder. Expected files:

| File | Shows | Status |
|------|-------|--------|
| `deepwhistle_overview.png` | The module running in PAMGuard (source spectrogram with model output + masked FFT spectrogram). The screenshot already captured can be used here. | needed |
| `deepwhistle_settings.png` | The Deep Whistle settings pane (close-up of the dialog). | needed |
| `deepwhistle_masked_spectrogram.png` | A masked FFT spectrogram showing only whistle contours. | needed |
| `deepwhistle_model_info.png` | The model information pop-up (description + paper link). | optional |
| `deepwhistle_datamodel.png` | The Add Modules menu or the data model chain. | optional |

Any additional figures can be added and referenced from the help file with `![](resources/<name>.png)`.
